## ¿Cómo sugerir un tema?

Palabras que no se repitan. Como sugerencia, puedes seleccionar un tema global y hacerlo más pequeño. Por ejemplo: tema global "Animal", tema del mes "delfin". Los temas globales son:
- Plantas
- Animales
- Planetas
- Comida
- Insectos
